Aula 7 - estudo de CSS

A sigla CSS é Cascading Style Sheeets é uma linguagem utilizada para estilizar paginas da web. Utilizado essa linguagem podemos definir

  • Cor de fundo, texto e elementos
  • fonte de tamanho das letras
  • Posicionamento de elementos dentro da página
  • efeitods visuais e animações

O CSS funciona em conjunto com HtML, emquanto o HTML diz que aparece, o CSS diz como deve aparecer.
